Функция bucket_sort. Реализует сортировку массива используя алгоритм сортировки BUCKET.

Parameters
[in]arr[]- Массив, который необходимо отсортировать.
[in]size- Размер передаваемого массива.
[in]rad- Параметр для получения n-го порядка числа.
bucket- Вектор для размещения корзин по индексу.
